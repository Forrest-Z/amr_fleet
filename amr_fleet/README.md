## Dependencies
* map_server
* amcl
* turtlebot3
* turtlebot_simulation
* turtlebot_msgs
* yaml-cpp

## Install packages dependencies
Run command from workspace root:
```rosdep install --from-paths src --ignore-src --rosdistro melodic -y```

Yaml-cpp:
```sudo apt install libyaml-cpp-dev```
